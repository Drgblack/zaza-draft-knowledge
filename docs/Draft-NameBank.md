\## 4. Usage Rules



\- Use names \*\*only in positive or neutral educational contexts\*\*  

\- Do not mix with surnames unless pseudonymised (e.g., “Alex B.”)  

\- Respect locale sensitivity when localising content (use equivalents from DE/FR/ES/IT name sets)  

\- Names are for \*\*testing, QA, and demo output only\*\* – not for live teacher data
